## 🤖 AI Orchestration & Agentic Workflows

I design and apply **agentic workflows** that treat AI as a first-class collaborator in software development — not just a code-completion tool.

### What I Do

- **Multi-agent orchestration** — structuring tasks across specialized AI agents for code generation, review, testing, and refactoring
- **Prompt engineering & context management** — crafting precise system prompts, memory strategies, and context windows to get reliable, production-grade output from LLMs
- **AI-augmented development pipelines** — embedding AI agents into coding workflows for rapid prototyping, scaffolding, debugging, and documentation
- **Tool-use & function-calling patterns** — building agentic loops where AI agents invoke tools, evaluate results, and iterate autonomously toward a goal
- **Human-in-the-loop design** — maintaining engineering oversight and quality gates while maximizing AI throughput
- **Custom instruction sets & agent configurations** — tailoring AI behavior to specific codebases, conventions, and team standards

### Why It Matters

AI orchestration lets me move faster across the full stack — from database schema design to pixel-perfect UI — while keeping code maintainable, secure, and well-tested. It's a force multiplier, not a shortcut.
